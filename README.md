# NotesOrganizer

Приложение для создания заметок, управления задачами, установки напоминаний и планирования дел.

## О проекте

NotesOrganizer — это удобное приложение, которое позволяет пользователям создавать текстовые заметки, задачи, события и напоминания. Система подходит для школьников, студентов и офисных работников, которым нужен простой и функциональный инструмент организации личной информации.

Проект создан в рамках учебной работы.

# Основные возможности

 Создание и редактирование текстовых заметок

 Категории и цветовые метки заметок

 Создание задач с дедлайнами

 Уведомления и напоминания

 Планировщик с календарём

 Поиск и фильтрация заметок

 Синхронизация данных с облаком

 Экспорт заметок (текст / JSON)

 Работа в offline-режиме
